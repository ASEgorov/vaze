## Основная информация

| Параметр | Значение |
|----------|----------|
| **Название** | Работа с библиотекой формул |
| **ID** | UC-006 |
| **Актер** | Пользователь |
| **Цель** | Просмотр, выбор, клонирование и добавление формул в библиотеку |
| **Предусловия** | Библиотека инициализирована предустановленными формулами |
| **Постусловия** | Пользователь выбрал формулу для редактирования или добавил свою |

## Основной поток (Basic Flow)

| Шаг | Актер | Система | Комментарий |
|-----|-------|---------|-------------|
| 1 | Открывает раздел «Библиотека формул» | Загружает каталог формул | Предустановленные + пользовательские |
| 2 | — | Отображает каталог карточками | Минимум 5 предустановленных формул |
| 3 | — | Отображает категории формул | Классические, Волнообразные, Спиральные, Геометрические, Абстрактные |
| 4 | Выбирает фильтр по категории | Фильтрует каталог по выбранной категории | Только карточки категории |
| 5 | Наводит курсор на карточку формулы | Показывает превью 3D-модели формулы | Мини-рендер или статичный SVG |
| 6 | Нажимает «Выбрать» на карточке | — | Выбор формулы |
| 7 | — | Загружает формулу и параметры в редактор | Связано с UC-001 |
| 8 | — | Обновляет 3D-превью с выбранной моделью | Связано с UC-003 |
| 9 | — | Показывает уведомление: `Формула {name} загружена` | Toast-уведомление |

## Альтернативные потоки (Alternative Flows)

### AF-1: Клонирование формулы
- **Шаг:** 5
- **Условие:** Пользователь нажимает «Клонировать» на карточке
- **Действие:** Система создаёт копию формулы с именем `Копия: {name}`, добавляет в «Мои формулы»
- **Возврат к:** Шаг 2 (обновлённый список)

### AF-2: Клонирование и переход к редактированию
- **Шаг:** 5
- **Условие:** Пользователь нажимает «Клонировать и редактировать»
- **Действие:** Система клонирует формулу, загружает в редактор, показывает UC-001
- **Возврат к:** Шаг 7

### AF-3: Добавление собственной формулы
- **Шаг:** 4
- **Условие:** Пользователь создаёт формулу в редакторе и нажимает «Сохранить в библиотеку»
- **Действие:** Система валидирует имя и формулу, добавляет в каталог выбранной категории
- **Возврат к:** Шаг 2 (обновлённый каталог с новой формулой)

### AF-4: Редактирование собственной формулы в библиотеке
- **Шаг:** 2
- **Условие:** Пользователь нажимает «Редактировать» на своей формуле
- **Действие:** Система загружает формулу в редактор
- **Возврат к:** Шаг 7

### AF-5: Поиск формулы в библиотеке
- **Шаг:** 1
- **Условие:** Пользователь использует строку поиска
- **Действие:** Система фильтрует каталог по названию формулы (partial match)
- **Возврат к:** Шаг 2 (результаты поиска)

### AF-6: Сортировка формул
- **Шаг:** 2
- **Условие:** Пользователь выбирает сортировку (по дате, по имени, по популярности)
- **Действие:** Система сортирует каталог по выбранному критерию
- **Возврат к:** Шаг 2

### AF-7: Просмотр подробной информации о формуле
- **Шаг:** 5
- **Условие:** Пользователь нажимает «Подробнее» на карточке
- **Действие:** Система показывает модальное окно: формула, параметры, описание, автор
- **Возврат к:** Шаг 5 (возврат к каталогу)

## Потоки ошибок (Error Flows)

### EF-1: Библиотека пуста
- **Шаг:** 1
- **Ошибка:** Нет предустановленных формул (ошибка загрузки)
- **Обработка:** Показать: `Библиотека пуста. Начните с создания собственной формулы.`
- **Возврат к:** — (перенаправление в редактор)

### EF-2: Формула удалена
- **Шаг:** 6
- **Условие:** Пользователь нажимает на формулу, но она была удалена
- **Обработка:** Показать: `Формула была удалена и недоступна.`
- **Возврат к:** Шаг 2 (обновлённый каталог)

### EF-3: Формула с таким именем уже существует
- **Шаг:** 9 (AF-3)
- **Условие:** Пользователь сохраняет формулу с дублирующим именем
- **Обработка:** Показать: `Формула с именем '{name}' уже существует. Заменить? Да / Нет`
- **Возврат к:** Шаг 9 (замена) или Шаг 4 (новое имя)

## Бизнес-правила

| ID | Правило | Описание |
|----|---------|----------|
| BP-01 | Минимум формул | Каталог содержит минимум 5 предустановленных формул |
| BP-02 | Категории | Классические, Волнообразные, Спиральные, Геометрические, Абстрактные |
| BP-03 | Имя клона | Формат: `Копия: {name}` |
| BP-04 | Приватность | Формулы пользователя приватны по умолчанию |
| BP-05 | Поиск | Partial match по названию формулы |
| BP-06 | Сортировка | По умолчанию: по дате добавления (новые первыми) |
