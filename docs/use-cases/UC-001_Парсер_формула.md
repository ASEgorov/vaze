## Основная информация

| Параметр | Значение |
|----------|----------|
| **Название** | Парсер формул |
| **ID** | UC-001 |
| **Актер** | Пользователь |
| **Цель** | Интерпретация математической формулы пользователя и вычисление значений |
| **Предусловия** | Пользователь ввёл текст формулы в редакторе |
| **Постусловия** | Система получила массив вычисленных точек для генерации геометрии |

## Основной поток (Basic Flow)

| Шаг | Актер | Система | Комментарий |
|-----|-------|---------|-------------|
| 1 | Вводит формулу `r = 5 + 2 * sin(h * 0.1)` | Принимает ввод и запускает валидацию | debounce ~200ms |
| 2 | — | Проверяет синтаксис формулы (скобки, операторы, функции) | Парсер выражения |
| 3 | — | Проверяет наличие допустимых переменных (`h`, `theta`) | whitelist переменных |
| 4 | — | Проверяет наличие допустимых функций (`sin`, `cos`, `pow` и т.д.) | whitelist функций |
| 5 | — | Если ошибок нет — компилирует формулу в исполняемый код | JIT-компиляция |
| 6 | — | Вычисляет формулу для диапазона h от 0 до maxHeight с шагом | Массив точек (r, h) |
| 7 | — | Возвращает массив точек системе генерации геометрии | 64–256 точек |
| 8 | — | Обновляет 3D-превью с новой моделью | Связано с UC-003 |

## Альтернативные потоки (Alternative Flows)

### AF-1: Ошибка синтаксиса
- **Шаг:** 2
- **Условие:** Формула содержит синтаксическую ошибку (незакрытая скобка, некорректный оператор)
- **Действие:** Система останавливает парсинг и возвращает ошибку с позицией
- **Возврат к:** Шаг 1 (пользователь исправляет)

### AF-2: Неизвестная функция
- **Шаг:** 4
- **Условие:** Формула содержит функцию, не входящую в whitelist
- **Действие:** Система возвращает ошибку со списком доступных функций
- **Возврат к:** Шаг 1 (пользователь заменяет функцию)

### AF-3: Ошибка вычисления — деление на ноль
- **Шаг:** 6
- **Условие:** При вычислении для некоторого значения h возникает деление на ноль
- **Действие:** Система пропускает этот шаг и показывает предупреждение о разрыве модели
- **Возврат к:** Шаг 7 (генерация с пропуском)

### AF-4: Формула возвращает NaN
- **Шаг:** 6
- **Условие:** Результат вычисления — NaN или Infinity
- **Действие:** Система показывает: `Результат вычисления некорректен. Проверьте формулу.`
- **Возврат к:** Шаг 1

## Потоки ошибок (Error Flows)

### EF-1: Парсер недоступен
- **Шаг:** 2
- **Ошибка:** Библиотека парсера (math.js) не загружена
- **Обработка:** Показать: `Инициализация парсера... Подождите.`
- **Возврат к:** Шаг 1

### EF-2: Переполнение стека
- **Шаг:** 5
- **Ошибка:** Слишком сложная формула (глубокая вложенность)
- **Обработка:** Показать: `Формула слишком сложная. Упростите выражение.`
- **Возврат к:** Шаг 1

## Бизнес-правила

| ID | Правило | Описание |
|----|---------|----------|
| BP-01 | Переменная h | Диапазон: 0 – maxHeight (мм) |
| BP-02 | Переменная theta | Диапазон: 0 – 2π (для спиральных ваз) |
| BP-03 | Доступные функции | sin, cos, tan, pow, abs, sqrt, log, exp, pi, e |
| BP-04 | Доступные операторы | +, -, *, /, ^ (степень) |
| BP-05 | Шаг вычисления | По умолчанию 1 мм (настраиваемый) |
| BP-06 | Лимит вычислений | Не более 10000 шагов для предотвращения зависаний |

## Данные и интерфейсы

### Входные данные
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| formula | string | Да | Текстовое выражение формулы |
| maxHeight | number | Да | Максимальная высота в мм |
| step | number | Нет | Шаг вычисления (мм), по умолчанию 1 |

### Выходные данные
| Параметр | Тип | Описание |
|----------|-----|----------|
| points | Point2D[] | Массив точек [{r, h}] для генерации поверхности |
| isValid | boolean | Результат валидации |
| errors | ValidationError[] | Список ошибок валидации (если есть) |
| warnings | Warning[] | Предупреждения (разрывы, NaN) |
