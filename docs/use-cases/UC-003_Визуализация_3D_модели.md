## Основная информация

| Параметр | Значение |
|----------|----------|
| **Название** | Визуализация 3D-модели |
| **ID** | UC-003 |
| **Актер** | Пользователь |
| **Цель** | Отображение 3D-модели вазы в браузере с интерактивным управлением камерой |
| **Предусловия** | Сцена инициализирована, геометрия вазы доступна (UC-002) |
| **Постусловия** | Пользователь видит и может взаимодействовать с 3D-моделью |

## Основной поток (Basic Flow)

| Шаг | Актер | Система | Комментарий |
|-----|-------|---------|-------------|
| 1 | — | Создаёт 3D-сцену (Scene), камеру (PerspectiveCamera) и рендерер (WebGLRenderer) | Инициализация при загрузке |
| 2 | — | Добавляет освещение (AmbientLight + DirectionalLight) | Два источника для реалистичного вида |
| 3 | — | Создаёт материал (MeshStandardMaterial или MeshPhysicalMaterial) | С возможностью изменения цвета/текстуры |
| 4 | — | Создаёт меш из геометрии вазы (BufferGeometry + Material) | Связано с UC-002 |
| 5 | — | Добавляет меш на сцену | Ваза центрирована по оси XZ |
| 6 | — | Подключает OrbitControls для управления камерой | Вращение, зум, панорамирование |
| 7 | — | Запускает цикл рендеринга (requestAnimationFrame) | 60 FPS при возможных условиях |
| 8 | Вращает камеру мышью | Обновляет позицию и target камеры | OrbitControls |
| 9 | Масштабирует колёсиком | Изменяет position.z (или zoom) камеры | Плавная анимация |
| 10 | — | При каждом кадре перерисовывает сцену | Рендер loop |

## Альтернативные потоки (Alternative Flows)

### AF-1: Обновление модели при изменении формулы
- **Шаг:** 4
- **Условие:** Пользователь изменил формулу, новая геометрия доступна
- **Действие:** Система удаляет старый меш, создаёт новый, обновляет сцену
- **Возврат к:** Шаг 4

### AF-2: Переключение режима отображения
- **Шаг:** 3
- **Условие:** Пользователь переключил режим отображения
- **Действие:** Система меняет material.side на BackSide (wireframe) или включил/выключил wireframe режим
- **Возврат к:** Шаг 3

### AF-3: Мобильное устройство (тач-управление)
- **Шаг:** 6
- **Условие:** Устройство поддерживает touch-события
- **Действие:** OrbitControls переключается на touch-режим: one-finger rotate, two-finger pan, pinch-zoom
- **Возврат к:** Шаг 6

### AF-4: Отображение размерной сетки и осей
- **Шаг:** 5
- **Условие:** Пользователь включил отображение осей/сетки
- **Действие:** Система добавляет GridHelper и AxesHelper на сцену
- **Возврат к:** Шаг 5

## Потоки ошибок (Error Flows)

### EF-1: WebGL не поддерживается
- **Шаг:** 1
- **Ошибка:** Браузер не поддерживает WebGL 2.0
- **Обработка:** Показать: `Ваш браузер не поддерживает WebGL. Используйте Chrome, Firefox или Edge.`
- **Возврат к:** — (приложение недоступно)

### EF-2: Нехватка GPU памяти
- **Шаг:** 7
- **Ошибка:** Ошибка выделения GPU-буферов для меша
- **Обработка:** Показать: `Недостаточно памяти GPU. Уменьшите детализацию модели.`
- **Возврат к:** Шаг 1 (с ограничением сегментов)

### EF-3: Потеря контекста WebGL
- **Шаг:** 7
- **Ошибка:** WebGL context lost (переключение вкладки, GPU сбой)
- **Обработка:** Подписаться на contextrestored и пересоздать сцену
- **Возврат к:** Шаг 1

## Бизнес-правила

| ID | Правило | Описание |
|----|---------|----------|
| BP-01 | WebGL | Минимальная поддержка WebGL 2.0 |
| BP-02 | Частота кадров | Целевая 60 FPS для моделей до 100K треугольников |
| BP-03 | Освещение | Минимум Ambient + Directional для корректного вида |
| BP-04 | Фон сцены | Светло-серый (#e0e0e0) или прозрачный |
| BP-05 | Адаптивность | Canvas занимает всю доступную область сцены |

## Данные и интерфейсы

### Входные данные
| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| geometry | BufferGeometry | Да | 3D-геометрия вазы от UC-002 |
| materialOptions | MaterialProps | Нет | Цвет, roughness, metalness |
| cameraPosition | Vector3 | Нет | Начальная позиция камеры |

### Выходные данные
| Параметр | Тип | Описание |
|----------|-----|----------|
| scene | Scene | Three.js сцена с вазой и окружением |
| camera | PerspectiveCamera | Перспективная камера |
| renderer | WebGLRenderer | Рендерер сцен |
| controls | OrbitControls | Контроллер камеры |
