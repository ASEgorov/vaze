## Описание

**Как** пользователь приложения VaseForge,  
**я хочу** задать максимальную высоту вазы в миллиметрах,  
**чтобы** формула вычисляла радиус для всей нужной мне модели.

## Контекст

Формула задаёт зависимость радиуса от высоты, но без задания диапазона (от 0 до какого значения?) система не может построить модель. Высота — ключевой параметр, определяющий физический размер вазы.

## Критерии приёмки (Acceptance Criteria)

### Сценарий 1: Установка высоты через поле ввода — Happy Path
- **Дано** пользователь ввёл формулу радиуса
- **Когда** пользователь вводит `150` в поле «Высота» и нажимает Enter
- **То** система обновляет диапазон вычисления формулы от h=0 до h=150

### Сценарий 2: Установка высоты через ползунок — Happy Path
- **Дано** пользователь видит ползунок высоты
- **Когда** пользователь перемещает ползунок к отметке 200 мм
- **То** система обновляет диапазон и перестраивает 3D-модель в реальном времени

### Сценарий 3: Ввод некорректного значения — Error Case
- **Дано** пользователь в фокусе поля «Высота»
- **Когда** пользователь вводит `abc` или `-50`
- **То** система подсвечивает поле красным и показывает: `Введите число от 10 до 1000 мм`

### Сценарий 4: Сброс к значению по умолчанию — Happy Path
- **Дано** пользователь изменил высоту
- **Когда** пользователь нажимает кнопку «Сброс» рядом с полем высоты
- **То** высота возвращается к `100 мм` (значение по умолчанию)

## Технические заметки

- Диапазон допустимых значений: 10–1000 мм
- Значение по умолчанию: 100 мм
- Шаг ползунка: 1 мм
- Формат ввода: целые числа и десятичные дроби
