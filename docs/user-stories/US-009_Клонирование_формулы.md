## Описание

**Как** пользователь приложения VaseForge,  
**я хочу** клонировать готовую формулу из библиотеки и изменить её,  
**чтобы** создать уникальную вазу на основе существующего дизайна без начала с нуля.

## Контекст

Пользователь может найти формулу, которая близка к нужному дизайну, но требует небольших корректировок. Клонирование позволяет быстро создать вариант без ввода формулы вручную.

## Критерии приёмки (Acceptance Criteria)

### Сценарий 1: Клонирование через контекстное меню — Happy Path
- **Дано** пользователь в каталоге формул
- **Когда** пользователь нажимает «⋮» на карточке и выбирает «Клонировать»
- **То** формула копируется в раздел «Мои формулы» с именем `Копия: [оригинальное_имя]`

### Сценарий 2: Клонирование и переход к редактированию — Happy Path
- **Дано** пользователь выбрал формулу для клонирования
- **Когда** пользователь нажимает «Клонировать и редактировать»
- **То** клон создаётся, редактор открывается сразу с клонированной формулой

### Сценарий 3: Клонирование без учётной записи — Happy Path
- **Дано** пользователь не авторизован (если будет авторизация)
- **Когда** пользователь нажимает «Клонировать»
- **То** клон сохраняется в локальное хранилище как черновик

### Сценарий 4: Клонирование несуществующей формулы — Error Case
- **Дано** оригинальная формула была удалена
- **Когда** пользователь нажимает «Клонировать»
- **То** система показывает: `Оригинальная формула была удалена и не может быть клонирована.`

## Технические заметки

- Клонирование: глубокая копия JSON-структуры формулы + параметров
- Имя клона: `Копия: {name}_{timestamp}` или `Копия: {name}_#N`
- Клон считается собственным проектом пользователя (можно редактировать, сохранять, удалять)
