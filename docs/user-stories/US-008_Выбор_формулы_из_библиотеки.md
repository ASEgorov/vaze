## Описание

**Как** пользователь приложения VaseForge,  
**я хочу** выбрать готовую формулу из библиотеки и увидеть её превью,  
**чтобы** быстро начать с базового дизайна и затем настроить его под себя.

## Контекст

Новые пользователи не знают, с чего начать. Библиотека формул — это точка входа: пользователь выбирает шаблон, видит результат и может модифицировать формулу. Это снижает порог входа и стимулирует эксперименты.

## Критерии приёмки (Acceptance Criteria)

### Сценарий 1: Просмотр каталога формул — Happy Path
- **Дано** пользователь открыл приложение
- **Когда** пользователь переходит в раздел «Библиотека формул»
- **То** отображается каталог с карточками готовых формул (минимум 5 типов)

### Сценарий 2: Предпросмотр формулы — Happy Path
- **Дано** пользователь в каталоге формул
- **Когда** пользователь наводит курсор на карточку «Коническая ваза»
- **То** справа отображается мини-превью 3D-модели этой формулы

### Сценарий 3: Выбор формулы для редактирования — Happy Path
- **Дано** пользователь в каталоге формул
- **Когда** пользователь нажимает «Выбрать» на карточке «Волнообразная ваза»
- **То** формула загружается в редактор, параметры заполняются значениями по умолчанию, отображается 3D-превью

### Сценарий 4: Фильтрация по категории — Happy Path
- **Дано** пользователь в каталоге формул
- **Когда** пользователь выбирает фильтр «Классические»
- **То** отображаются только формулы категории «Классические»

### Сценарий 5: Выбор пустой формулы — Happy Path
- **Дано** пользователь в каталоге формул
- **Когда** пользователь нажимает «Начать с нуля»
- **То** открывается пустой редактор формулы с примером `r = 5 + 2 * sin(h * 0.1)` в качестве подсказки

## Технические заметки

- Хранение формул: JSON-коллекция в коде или удалённое хранилище
- Структура формулы: `{ id, name, category, formula, params: { maxHeight, segments }, previewMesh }`
- Категории: Классические, Волнообразные, Спиральные, Геометрические, Абстрактные
- Превью: предгенерированные меши или лёгкие версии для быстрого рендера
