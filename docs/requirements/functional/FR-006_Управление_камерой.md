## Описание

Система обеспечивает интерактивное управление камерой 3D-сцены: вращение, масштабирование, панорамирование и сброс позиции с поддержкой мыши и touch-устройств.

## Входящие данные

| Параметр | Тип | Описание | Обязательный |
|----------|-----|----------|--------------|
| container | HTMLDivElement | Контейнер 3D-сцены | Да |
| camera | PerspectiveCamera | Камера 3D-сцены | Да |
| renderer | WebGLRenderer | WebGL-рендерер | Да |

## Выходные данные

| Параметр | Тип | Описание |
|----------|-----|----------|
| controls | OrbitControls | Настроенный контроллер камеры |
| cameraPosition | Vector3 | Текущая позиция камеры |
| cameraTarget | Vector3 | Текущий target камеры |
| isTouchDevice | boolean | Флаг touch-устройства |

## Логика обработки

1. Определение типа устройства:
   - `isTouchDevice = 'ontouchstart' in window || navigator.maxTouchPoints > 0`
2. Инициализация OrbitControls:
   - `controls = new OrbitControls(camera, renderer.domElement)`
3. Настройка параметров управления:
   - EnableDamping: true (инерция при вращении)
   - DampingFactor: 0.08 (степень затухания)
   - EnableZoom: true
   - ZoomSpeed: 1.0
   - EnablePan: true
   - PanSpeed: 0.8
   - RotateSpeed: 0.6
4. Ограничения камеры:
   - Min distance: 10 (максимальный zoom in)
   - Max distance: 5000 (максимальный zoom out)
   - Min polar angle: 0 (вид сверху)
   - Max polar angle: PI (вид снизу)
   - Enable pan bounds: true
   - Pan bounds: bounding box модели + отступ 10%
5. Настройка touch-управления (если isTouchDevice):
   - one-finger touch → rotate
   - two-finger touch → pan
   - pinch (two-finger spread) → zoom
6. Обработка событий мыши:
   - mousedown (LMB) → начало вращения
   - mousemove → вращение камеры вокруг target
   - wheel → масштабирование
   - mousedown (RMB) → начало панорамирования
   - mousemove → панорамирование
7. Обработка touch-событий:
   - touchstart → запоминание позиций пальцев
   - touchmove → обработка rotate / pan / zoom
   - touchend → сброс состояния
8. Поддержка клавиатуры (опционально):
   - Стрелки → вращение
   - + / - → масштабирование
   - Home → сброс камеры
9. Slerp-интерполяция при сбросе:
   - Плавный переход от текущей позиции к начальной
   - Длительность: 500ms

## Обработка ошибок

| Код ошибки | Ситуация | Действие системы | Сообщение пользователю |
|------------|----------|------------------|------------------------|
| ERR-501 | OrbitControls не доступен | Остановка на шаге 2 | `Ошибка инициализации управления камерой.` |
| ERR-502 | Контейнер не найден | Остановка на шаге 1 | `Ошибка: область 3D-сцены не найдена.` |
| ERR-503 | Touch-события конфликтуют | На шаге 6 | `Отмена стандартных жестов браузера` (автоматически) |
