## Описание

Система отображает 3D-модель вазы в браузере с использованием WebGL (Three.js / React Three Fiber), обеспечивая корректное освещение, материалы и адаптивный рендеринг.

## Входящие данные

| Параметр | Тип | Описание | Обязательный |
|----------|-----|----------|--------------|
| geometry | BufferGeometry | 3D-геометрия от FR-003 | Да |
| materialOptions | MaterialOpts | Опции материала (цвет, roughness) | Нет |
| cameraConfig | CameraConfig | Настройки камеры | Нет |
| sceneConfig | SceneConfig | Настройки фона, освещения | Нет |

## Выходные данные

| Параметр | Тип | Описание |
|----------|-----|----------|
| scene | Scene | Инициализированная Three.js сцена |
| camera | PerspectiveCamera | Перспективная камера |
| renderer | WebGLRenderer | Настроенный WebGL-рендерер |
| controls | OrbitControls | Контроллер камеры (если включён) |
| animationId | number | ID animation frame для очистки |

## Логика обработки

1. Проверка поддержки WebGL:
   - Попытка получить `gl = canvas.getContext('webgl2')`
   - Если не удалось: показать ошибку и отключить 3D-секцию
2. Создание сцены: `scene = new THREE.Scene()`
3. Настройка фона: `scene.background = new THREE.Color(sceneConfig.background || '#e0e0e0')`
4. Создание камеры:
   - `camera = new THREE.PerspectiveCamera(45, aspect, 0.1, 10000)`
   - Позиция по умолчанию: `(0, maxHeight/2, maxHeight * 2)`
   - Target: центр bounding box модели
5. Создание рендерера:
   - `renderer = new THREE.WebGLRenderer({ antialias: true, alpha: false })`
   - Размер: `renderer.setSize(containerWidth, containerHeight)`
   - Pixel ratio: `renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2))`
6. Настройка освещения:
   - AmbientLight: intensity 0.5, color 0xffffff
   - DirectionalLight: intensity 0.8, position `(5, 10, 7)`
   - (Опционально) HemisphereLight: intensity 0.3
7. Создание материала:
   - `material = new THREE.MeshStandardMaterial(materialOptions || { color: 0x4a90d9, roughness: 0.7, metalness: 0.1 })`
8. Создание меша: `mesh = new THREE.Mesh(geometry, material)`
9. Добавление меша на сцену: `scene.add(mesh)`
10. Отображение осей (опционально): `scene.add(new THREE.AxesHelper(50))`
11. Отображение сетки (опционально): `scene.add(new THREE.GridHelper(maxHeight, 10))`
12. Инициализация OrbitControls:
    - `controls = new OrbitControls(camera, renderer.domElement)`
    - EnableDamping: true
    - Min distance: 10, max distance: 5000
    - Min/Polar angle: 0, PI (полный обзор)
13. Запуск animation loop:
    - `renderer.setAnimationLoop(render)`
    - render(): `controls.update()`, `renderer.render(scene, camera)`
14. Обработка resize: подписка на `window.resize`, обновление камеры и renderer

## Обработка ошибок

| Код ошибки | Ситуация | Действие системы | Сообщение пользователю |
|------------|----------|------------------|------------------------|
| ERR-301 | WebGL не поддерживается | Остановка на шаге 1 | `Ваш браузер не поддерживает WebGL. Используйте Chrome, Firefox или Edge.` |
| ERR-302 | Недостаточно GPU памяти | На шаге 8 | `Недостаточно памяти GPU. Уменьшите детализацию.` |
| ERR-303 | WebGL context lost | Во время рендеринга | `Перезапуск 3D-сцены...` (автовосстановление) |
| ERR-304 | Пустая геометрия | На шаге 8 | `Нет геометрии для отображения.` |
