## Описание

Система принимает текстовое выражение формулы от пользователя, проверяет его синтаксис и семантику, и возвращает результат валидации с детальной информацией об ошибках.

## Входящие данные

| Параметр | Тип | Описание | Обязательный |
|----------|-----|----------|--------------|
| formulaText | string | Текстовое выражение формулы | Да |
| variables | string[] | Доступные переменные (h, theta) | Да |
| functions | string[] | Доступные функции (sin, cos, pow и т.д.) | Да |
| operators | string[] | Доступные операторы (+, -, *, /, ^) | Да |

## Выходные данные

| Параметр | Тип | Описание |
|----------|-----|----------|
| isValid | boolean | Результат валидации |
| parsedExpression | AST \| null | Дерево разбора формулы (если валидна) |
| errors | ValidationError[] | Массив ошибок с позициями и сообщениями |
| warnings | Warning[] | Предупреждения (не критические) |

## Логика обработки

1. Получение формулы от пользователя (с debounce ~200ms)
2. Проверка баланса скобок: каждая открывающая `(` должна иметь закрывающую `)`
3. Лексический анализ: разделение на токены (числа, переменные, функции, операторы)
4. Проверка токенов:
   - Числа: валидация формата (целые, десятичные, научная нотация)
   - Переменные: проверка по whitelist (h, theta)
   - Функции: проверка по whitelist (sin, cos, tan, pow, abs, sqrt, log, exp, pi, e)
   - Операторы: проверка по whitelist (+, -, *, /, ^)
5. Синтаксический анализ (постфиксная нотация / Shunting-Yard algorithm)
6. Проверка семантики:
   - Корректность количества аргументов у функций (sin(x) — 1, pow(x,y) — 2)
   - Отсутствие деления на константу 0
7. Если ошибок нет — компиляция в AST (Abstract Syntax Tree)
8. Возврат результатов валидации

## Обработка ошибок

| Код ошибки | Ситуация | Действие системы | Сообщение пользователю |
|------------|----------|------------------|------------------------|
| ERR-001 | Незакрытая скобка | Остановка на шаге 2 | `Незакрытая скобка на позиции ${pos}` |
| ERR-002 | Неизвестная переменная | Остановка на шаге 4 | `Неизвестная переменная '${token}'. Используйте: h, theta` |
| ERR-003 | Неизвестная функция | Остановка на шаге 4 | `Функция '${token}' не найдена. Доступные: ${availableList}` |
| ERR-004 | Неправильное количество аргументов | Остановка на шаге 6 | `Функция '${func}' ожидает ${expected} аргументов, получено ${actual}` |
| ERR-005 | Некорректный синтаксис | Остановка на шаге 5 | `Ошибка синтаксиса на позиции ${pos}: ${detail}` |
| ERR-006 | Пустая формула | Остановка на шаге 1 | `Введите формулу` |
