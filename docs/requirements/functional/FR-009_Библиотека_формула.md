## Описание

Система предоставляет каталог предустановленных формул с категориями, фильтрацией, поиском и предпросмотром 3D-модели для каждой формулы.

## Входящие данные

| Параметр | Тип | Описание | Обязательный |
|----------|-----|----------|--------------|
| categoryFilter | string | Фильтр по категории | Нет |
| searchQuery | string | Текст поиска | Нет |
| sortBy | string | Сортировка: `date`, `name`, `popular` | Нет, по умолчанию `date` |

## Выходные данные

| Параметр | Тип | Описание |
|----------|-----|----------|
| formulas | FormulaCard[] | Отфильтрованный список формул |
| categories | string[] | Список всех категорий |
| totalCount | number | Общее количество формул |
| filteredCount | number | Количество после фильтрации |

## Логика обработки

1. Загрузка библиотеки формул:
   - Предустановленные формулы загружаются при инициализации приложения
   - Пользовательские формулы загружаются из хранилища
2. Категории предустановленных формул:
   - Классические: конус, цилиндр, усечённый конус
   - Волнообразные: синусоида, косинусоида, комбинированная
   - Спиральные: винт, спираль, двойная спираль
   - Геометрические: кубическая, ступенчатая, звёздчатая
   - Абстрактные: случайная генерация, Lissajous
3. Применение фильтрации:
   - По категории: `formulas.filter(f => f.category === categoryFilter)`
   - По поиску: `formulas.filter(f => f.name.includes(searchQuery))`
   - Комбинирование фильтров
4. Применение сортировки:
   - По дате: `formulas.sort((a, b) => b.createdAt - a.createdAt)`
   - По имени: `formulas.sort((a, b) => a.name.localeCompare(b.name))`
5. Генерация превью:
   - Для каждой карточки вычислить мини-геометрию с низким segments (16)
   - Рендер превью в offscreen canvas или SVG
   - Кэшировать превью для повторного использования
6. Отображение каталога:
   - Сетка карточек (responsive grid)
   - Каждая карточка: превью, имя, категория, описание, кнопки «Выбрать» / «Клонировать»
7. Обработка выбора:
   - При клике «Выбрать»: загрузить формулу в редактор (FR-001 → FR-003)
   - При клике «Клонировать»: создать копию в «Мои проекты»

## Обработка ошибок

| Код ошибки | Ситуация | Действие системы | Сообщение пользователю |
|------------|----------|------------------|------------------------|
| ERR-801 | Пустая библиотека | На шаге 1 | `Библиотека пуста.` |
| ERR-802 | Ошибка генерации превью | На шаге 5 | `Ошибка предпросмотра. Формула будет доступна без превью.` |
| ERR-803 | Повреждённая формула в библиотеке | На шаге 3 | `Формула повреждена и удалена из каталога.` |
