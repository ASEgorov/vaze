## Описание

Система создаёт глубокую копию (клон) формулы из библиотеки, добавляет её в пользовательские проекты и при необходимости загружает в редактор.

## Входящие данные

| Параметр | Тип | Описание | Обязательный |
|----------|-----|----------|--------------|
| sourceFormula | Formula | Исходная формула для клонирования | Да |
| cloneName | string | Имя клона | Нет, по умолчанию `Копия: {sourceName}` |
| openInEditor | boolean | Открыть клон в редакторе | Нет, по умолчанию false |

## Выходные данные

| Параметр | Тип | Описание |
|----------|-----|----------|
| clonedFormula | Formula | Склонированная формула |
| clonedFormulaId | string | Уникальный ID клона |
| isSuccess | boolean | Результат клонирования |
| errorMessage | string | Сообщение об ошибке |

## Логика обработки

1. Валидация входных данных:
   - sourceFormula не null
   - sourceFormula.text — валидная строка формулы
2. Генерация имени клона:
   - Если cloneName не указано: `cloneName = 'Копия: ' + sourceFormula.name`
   - Проверка уникальности имени в хранилище пользователя:
     - Если дубликат: добавить индекс `cloneName = cloneName + '_' + Date.now()`
3. Глубокое копирование формулы:
   - `cloned = JSON.parse(JSON.stringify(sourceFormula))`
   - `cloned.id = generateUUID()`
   - `cloned.name = cloneName`
   - `cloned.isClone = true`
   - `cloned.sourceId = sourceFormula.id`
   - `cloned.createdAt = now()`
   - `cloned.updatedAt = now()`
4. Сохранение клона в хранилище:
   - Вызвать FR-007 (сохранение проекта) с данными клона
   - Если сохранение неудачно: вернуть ошибку
5. Если openInEditor === true:
   - Восстановить состояние редактора данными клона (FR-008)
   - Обновить 3D-превью (FR-004)
6. Обновить каталог библиотеки:
   - Добавить клон в список пользовательских формул
   - Обновить UI каталога
7. Показ уведомления: `Формула '{cloneName}' склонирована`
8. Возврат результатов

## Обработка ошибок

| Код ошибки | Ситуация | Действие системы | Сообщение пользователю |
|------------|----------|------------------|------------------------|
| ERR-901 | Пустой источник | Остановка на шаге 1 | `Невозможно клонировать: источник не указан.` |
| ERR-902 | Хранилище заполнено | На шаге 4 | `Хранилище заполнено. Удалите проекты.` |
| ERR-903 | Невалидная формула | На шаге 1 | `Исходная формула невалидна и не может быть склонирована.` |
