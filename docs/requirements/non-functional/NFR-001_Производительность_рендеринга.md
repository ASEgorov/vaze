## Описание

Система обеспечивает плавный рендеринг 3D-сцены с частотой не менее 30 FPS при интерактивном взаимодействии пользователя с моделью вазы, включая вращение, масштабирование и обновление геометрии при изменении формулы.

## Контекст

Производительность визуализации критична для UX: лаги и «тормоза» при вращении модели или обновлении после изменения формулы вызывают раздражение. Пользователь ожидает отзывчивого интерфейса, как в профессиональных 3D-редакторах.

## Метрики

| Метрика | Целевое значение | Измерение |
|---------|-----------------|-----------|
| FPS (idle) | ≥ 55 FPS | FPS counter при простое |
| FPS (interact) | ≥ 30 FPS | FPS counter при вращении/зуме |
| Время обновления модели | ≤ 500 мс | Time from formula change to rendered |
| Время загрузки сцены | ≤ 2 сек | Time from page load to interactive |
| Время экспорта STL | ≤ 3 сек для моделей до 100K тр | Performance API |
| Размер bundle (gzip) | ≤ 350 КБ | Bundle analyzer |

## Условия выполнения

- Измерение при стандартных условиях: современный браузер, 8+ ГБ RAM, GPU с WebGL 2.0
- Тестовые модели: от простой (1K тр) до сложной (500K тр)
- Тестирование на типичных устройствах: десктоп, планшет, мобильный
