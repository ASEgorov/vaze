## Описание

Система корректно работает во всех современных браузерах, поддерживающих WebGL 2.0, без необходимости установки дополнительных плагинов или расширений.

## Контекст

Пользователи VaseForge работают с разных устройств и браузеров. Приложение должно обеспечивать единообразный опыт на Chrome, Firefox, Safari и Edge, без «сломанного» UI или неработающей 3D-сцены.

## Метрики

| Метрика | Целевое значение | Измерение |
|---------|-----------------|-----------|
| Поддерживаемые браузеры | Chrome 90+, Firefox 88+, Safari 14+, Edge 90+ | Browser testing |
| WebGL 2.0 поддержка | ≥ 95% целевых пользователей | stats.gl |
| CSS Grid/Flexbox | Корректное отображение на всех браузерах | Visual regression |
| Touch-события | Корректная работа на iOS Safari и Android Chrome | Mobile testing |

## Условия выполнения

- Тестирование на последних 2 стабильных версиях каждого браузера
- Тестирование на десктопных и мобильных устройствах
- Проверка на устройствах с и без GPU ( integrated vs discrete)
