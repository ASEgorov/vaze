## Описание

Система поддерживает PWA-функциональность (установка на устройство, работа offline) и обеспечивает базовую доступность (accessibility) для пользователей с ограниченными возможностями.

## Контекст

PWA позволяет пользователям устанавливать приложение на домашний экран и использовать его без браузера. Offline-режим критичен для работы в местах с нестабильным интернетом. Accessibility обеспечивает inclusivity.

## Метрики

| Метрика | Целевое значение | Измерение |
|---------|-----------------|-----------|
| Lighthouse PWA score | ≥ 90 | Lighthouse audit |
| Lighthouse A11y score | ≥ 85 | Lighthouse audit |
| Работа offline | Базовый редактор + сохранение | Offline testing |
| Service Worker cache | Библиотека + ключевые данные | Cache size monitoring |
| Keyboard navigation | Все интерактивные элементы доступны | Keyboard audit |
| Screen reader support | ARIA-метки для всех элементов | Screen reader testing |

## Условия выполнения

- Тестирование Lighthouse на desktop и mobile
- Проверка работы без интернета (отключение network)
- Тестирование с VoiceOver (macOS/iOS) и NVDA (Windows)
- Проверка навигации только клавиатурой (Tab, Enter, Escape)
