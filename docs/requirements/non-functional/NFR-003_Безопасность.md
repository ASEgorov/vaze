## Описание

Система защищает пользователя от вредоносных формул и обеспечивает безопасное выполнение пользовательского кода в изолированной среде вычислений. Формулы пользователей не должны иметь доступа к DOM, сети или файловой системе.

## Контекст

Поскольку система исполняет пользовательские выражения (формулы), существует риск выполнения вредоносного кода. Необходимо обеспечить безопасность без блокировки легитимных математических операций.

## Метрики

| Метрика | Целевое значение | Измерение |
|---------|-----------------|-----------|
| Время исполнения формулы | ≤ 1 сек на выражение | Timeout enforcement |
| Количество разрешённых операций | ≤ 10000 на вычисление | Operation counter |
| Изоляция среды выполнения | 100% — нет доступа к window/document | Code audit |
| Защита от DoS | Все формулы завершаются или таймаутятся | Stress testing |

## Условия выполнения

- Тестирование с вредоносными формулами: рекурсия, бесконечные циклы, доступ к DOM
- Проверка whitelist функций и переменных
- Валидация длины формулы (не более 500 символов)
