## Описание

Пользователь задаёт параметрические формулы (радиус сечения, высота, угол поворота) и система автоматически строит 3D-модель вазы на основе этих формул.

## Контекст

Это ядро продукта — без генерации геометрии по формулам приложение теряет смысл. Пользователь описывает форму вазы через математические выражения, а система интерпретирует их и визуализирует.

## Критерии приёмки

- [ ] Пользователь может ввести формулу для радиуса сечения в зависимости от высоты
- [ ] Формула поддерживают переменные: h (высота), θ (угол), r (радиус)
- [ ] Формула поддерживают базовые функции: sin, cos, tan, pow, abs, sqrt
- [ ] Система корректно строит модель для валидных формул
- [ ] Пользователь может задать диапазон значений параметра (например, h от 0 до 100 мм)
