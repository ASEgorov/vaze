## Описание

Пользователь может экспортировать готовую 3D-модель вазы в формате STL (стереолитографии) для последующей 3D-печати.

## Контекст

STL — стандартный формат для 3D-печати. Экспорт — финальный шаг рабочего процесса: пользователь满意 моделью → скачивает STL → загружает в слайсер для печати.

## Критерии приёмки

- [ ] Экспорт поддерживает бинарный и ASCII форматы STL
- [ ] Экспортируемая модель соответствует масштабу, указанному пользователем (в мм)
- [ ] Файл STL открывается в стандартных слайсерах (Cura, PrusaSlicer и т.д.)
- [ ] Пользователь может скачать файл по нажатию одной кнопки
- [ ] При ошибках генерации (невалидная геометрия) отображается понятное сообщение
