## Описание

Пользователь может сохранять текущую конфигурацию проекта (формулы, параметры вазы, настройки визуализации) и загружать её позже для продолжения работы.

## Контекст

Пользователи могут работать над проектами длительное время, прерываться, возвращаться. Сохранение позволяет не терять прогресс и делиться проектами с другими.

## Критерии приёмки

- [ ] Пользователь может сохранить проект с именем и описанием
- [ ] Сохраняются все формулы, параметры, настройки камеры и масштаба
- [ ] Пользователь может загрузить ранее сохранённый проект
- [ ] Сохранённые проекты доступны в списке/библиотеке
- [ ] Пользователь может переименовывать и удалять сохранённые проекты
