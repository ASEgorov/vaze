## Описание

Система предоставляет библиотеку готовых формул и шаблонов ваз, которые пользователь может выбрать, посмотреть, модифицировать и использовать как основу для собственных дизайнов.

## Контекст

Новые пользователи могут не знать, с чего начать. Библиотека формул служит точкой входа — пользователь выбирает готовую форму (коническая, сферическая, волнообразная и т.д.), видит результат и может настроить параметры под себя. Это снижает порог входа и стимулирует эксперименты.

## Критерии приёмки

- [ ] Система поставляется с предустановленной коллекцией формул (минимум 5 типов ваз)
- [ ] Пользователь может просматривать превью каждой формулы из библиотеки
- [ ] Пользователь может выбрать формулу и открыть её для редактирования
- [ ] Пользователь может создать новую формулу на основе существующей (клонировать)
- [ ] Пользователь может добавлять собственные формулы в библиотеку
- [ ] Формулы в библиотеке категоризированы (форма, стиль, сложность)
