## Описание

Пользователь видит обновляемое в реальном времени 3D-превью модели, которое динамически изменяется при редактировании формул и параметров вазы.

## Контекст

Пользователь должен мгновенно видеть результат своих изменений без необходимости ждать перерасчёта или экспорта. Это позволяет итеративно подбирать формулу и получать визуальную обратную связь.

## Критерии приёмки

- [ ] 3D-превью обновляется при изменении формулы/параметра без перезагрузки страницы
- [ ] Пользователь может вращать, масштабировать и перемещать камеру 3D-сцены
- [ ] Визуализация поддерживает отображение осей координат и размерной сетки
- [ ] Задержка обновления при изменении формулы не превышает 500 мс
- [ ] Модель отображается корректно при различных комбинациях формул
