## Описание

Реализовать пользовательскую обработку ошибок и валидации (BR-005, FR-001): подсветка ошибок в поле формулы, тосты с сообщениями, fallback при сбоях WebGL.

## Подзадачи

- [ ] Создать `src/components/ErrorToast.tsx` — компонент тост-уведомлений
- [ ] Создать `src/hooks/useToast.ts` — хук для управления тостами (add, remove, clear)
- [ ] Интегрировать валидацию в formula input:
  - Красная подсветка при ошибке синтаксиса
  - Тултип с сообщением об ошибке при hover
  - Цветовая индикация: OK (зелёный), Warning (жёлтый), Error (красный)
- [ ] Добавить обработчик потери WebGL контекста:
  - `<Canvas onError>` — показать fallback UI
  - Автопересоздание контекста
- [ ] Создать `src/components/ValidationBadge.tsx` — индикатор статуса формулы
- [ ] Graceful degradation: если генерация не удалась — показать placeholder с сообщением
