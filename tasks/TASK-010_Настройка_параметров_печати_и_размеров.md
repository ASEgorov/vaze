## Описание

Реализовать отображение размеров модели (FR-013, BR-006) и настройку параметров печати (FR-012): ориентация, масштаб, слой, заполнение.

## Подзадачи

- [ ] Создать `src/components/ModelInfo.tsx`:
  - Отображение: ширина, глубина, высота, объём, площадь, кол-во тр
  - Обновление при изменении модели
  - Иконки + единицы измерения (мм, мм³, мм²)
- [ ] Создать `src/stores/printStore.ts` — Zustand-стор для настроек печати
- [ ] Реализовать параметры:
  - Ориентация: base (на дне), side (на боку), custom (угол)
  - Масштаб печати: 0.5, 0.75, 1.0, 1.25, 1.5, 2.0
  - Слой (высота слоя): 0.1, 0.15, 0.2, 0.3 мм
  - Заполнение (плотность): 10%, 20%, 50%, 100%
- [ ] Создать `src/components/PrintSettings.tsx`:
  - Dropdown для ориентации
  - Radio buttons для масштаба
  - Slider для толщины слоя
  - Progress bar для заполнения
- [ ] Показать ожидаемое время печати на основе объёма и скорости принтера
