## Описание

Провести интеграционное тестирование всех компонентов, проверку на совместимость с браузерами, тестирование производительности и UX.

## Подзадачи

- [ ] Настроить Vitest + React Testing Library
- [ ] Написать интеграционные тесты:
  - Pipeline: formula input → parse → geometry → mesh → render
  - Save → reload → verify data integrity
  - Export → verify STL file opens in slicer
  - Library: add → clone → load → delete
- [ ] Кросс-браузерное тестирование:
  - Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  - Desktop + mobile (iOS Safari, Android Chrome)
- [ ] Тестирование производительности:
  - FPS benchmarks (idle + interact)
  - Model update time
  - Bundle size verification
- [ ] UX-тестирование:
  - Время до первого результата ≤ 30 сек
  - Количество кликов до экспорта ≤ 5
  - Ошибки понятны пользователю
- [ ] Баг-репортинг и фиксы
