## Описание

Реализовать экспорт 3D-модели в бинарный формат STL (ADR-005). Функция должна принимать BufferGeometry и возвращать Blob для скачивания.

## Подзадачи

- [ ] Создать `src/utils/exportSTL.ts`
- [ ] Реализовать `exportToBinarySTL(geometry: BufferGeometry): Blob`
- [ ] Реализовать `exportToAsciiSTL(geometry: BufferGeometry): Blob` (опционально)
- [ ] Бинарный формат: 80-byte header + uint32 triangle count + 50-byte triangle records
- [ ] Каждая вершина треугольника: normal (12 bytes) + 3 vertices (36 bytes) + attribute byte count (2 bytes)
- [ ] Float32 little-endian порядок
- [ ] Создать `src/hooks/useSTLExport.ts` — хук с кнопкой экспорта
- [ ] Добавить UX: кнопка «Export STL», spinner во время экспорта, toast-уведомление об успехе/ошибке
- [ ] Юнит-тесты: проверка размера файла, корректность бинарного формата

## Критерии готовности (Definition of Done)

- [ ] Binary STL генерируется корректно
- [ ] Файл открывается в Cura / PrusaSlicer / Bambu Studio
- [ ] ExportToAscii работает (опционально)
- [ ] UX: кнопка, spinner, notification
- [ ] Размер файла соответствует ожиданиям (~50 bytes × triangles)
- [ ] Время экспорта ≤ 3 сек для моделей до 100K тр (NFR-001)
- [ ] Юнит-тесты проходят
- [ ] Code review проведён
